This is a paragraph, This is where the bulk of the content in a webpage goes, this is where arcticles are written, facts are given, reaserch is shared and much more. this is made using the "p" tag.

Text can also be made smaller than the default font size by using the tag "small"

Text can also be made more important looking than the default size using the tag "strong"

Text can also be made bold by using the tag "b"

-Text can also just go by itself in the body of a HTML page , it does not need to be in a header or paragraph.
